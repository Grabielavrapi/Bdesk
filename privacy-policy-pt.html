Nossa Política de Privacidade

Informações Sobre o Gerenciamento de Dados Pessoais por Solicitação de Informação

A empresa InterMedia, na qualidade de controladora de dados, fornece as seguintes informações aos clientes que acessam seu site https://bookersdesk.com/ sobre o processamento de seus dados:

Dados Pessoais Coletados

O BookersDesk lidará com os seguintes dados, fornecidos diretamente pela parte interessada, preenchendo os formulários no site e/ou enviando e-mails:

  1.  Dados necessários para entrar em contato com o cliente (por exemplo, nome, sobrenome, número de telefone celular, endereço de e-mail).
  2.  Dados fornecidos pelo cliente para explicar melhor suas solicitações contidas na mensagem, caso possam ser considerados dados pessoais.

Coleta de Dados

A coleta e o processamento de dados pessoais têm como único objetivo gerenciar o relacionamento com o cliente. Em particular, os dados serão processados para os seguintes fins:

  1.  Entrar em contato com o cliente através do canal indicado por ele e fornecer informações sobre as atividades do BookersDesk, sobre os serviços oferecidos.
  2.  Ativar uma conta de teste para avaliar os serviços oferecidos pelo BookersDesk.

Fornecimento Opcional de Dados e Consentimento para seu Processamento - Consequências da Recusa

A comunicação de seus dados ao BookersDesk, preenchendo os formulários ou enviando e-mails, ocorre por decisão espontânea do cliente.

Os dados indicados como obrigatórios no formulário são necessários para entrar em contato com o cliente e fornecer respostas relacionadas às suas necessidades. Se essas informações não forem fornecidas, o formulário não será enviado ao BookersDesk. Se o cliente interagir por e-mail, ele é livre para comunicar as informações e dados que considerar mais adequados; caso nenhum dado de contato seja fornecido, o BookersDesk usará o endereço de e-mail do remetente para esse fim.

Comunicação e Divulgação de Dados

Para fornecer informações ou um orçamento, os dados fornecidos pelos clientes serão processados pela equipe do BookersDesk dedicada à atividade comercial. Em todos os casos, os dados não serão divulgados a terceiros.

Método de Conservação, Duração e Tratamento de Dados Pessoais

Os dados arquivados serão mantidos por um período de um ano pelo BookersDesk, após o qual serão excluídos. Os dados não serão processados por meio de processos automatizados de tomada de decisão. Os dados nunca serão exportados para fora da Área Econômica Europeia. Os dados não serão processados por processos automatizados, exceto pelos processos relacionados à anonimização/exclusão em massa de dados, que são ativados periodicamente e cobrem todos os dados que possuem determinadas características. A cessação do tratamento resultará na exclusão dos dados armazenados em formato eletrônico.

Direitos do Titular dos Dados

  1.  Os candidatos têm o direito de solicitar ao BookersDesk, a qualquer momento, o acesso aos seus dados pessoais, a retificação ou exclusão deles, ou de se opor ao seu processamento.
  2.  As solicitações podem ser direcionadas ao BookersDesk usando o formulário de contato adequado.

Em qualquer caso, os candidatos sempre têm o direito de apresentar uma reclamação à autoridade supervisora competente se considerarem que o processamento de seus dados é contrário à legislação aplicável.

Controlador e Processador de Dados

O controlador de dados é o BookersDesk e o processador de dados é BookersDesk.
