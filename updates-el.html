Ενημερώσεις λογισμικού

Ιανουάριος 31, 2025

Εκθέσεις Ελλείμματος Εσόδων για Κρατήσεις

Προσθέσαμε μια νέα δυνατότητα που επιτρέπει τη δημιουργία μηνιαίων εκθέσεων εσόδων για κρατήσεις, παρέχοντας μια σαφή εικόνα των εσόδων και των τάσεων με την πάροδο του χρόνου.

Ιανουάριος 27, 2025

Το Ημερολόγιο Εμφανίζει Όλα τα Ακίνητα σε Μια Προβολή

Τώρα μπορείτε να δείτε όλα τα ακίνητά σας σε ένα μόνο ημερολόγιο, απλοποιώντας τη διαχείριση πολλαπλών τοποθεσιών και βελτιώνοντας τη ροή εργασίας σας.

Ιανουάριος 21, 2025

Προστέθηκαν Νέες Γλώσσες στο BookersDesk

Το BookersDesk υποστηρίζει τώρα νέες γλώσσες, όπως Ιταλικά, Ελληνικά, Ισπανικά, Γαλλικά και Γερμανικά, προσφέροντας περισσότερες επιλογές για διεθνείς χρήστες.

Δεκέμβριος 23, 2024

Η API του Hotelbeds Ενεργοποιήθηκε στην Παραγωγή

Η API του Hotelbeds έχει ενσωματωθεί με επιτυχία και είναι πλέον ενεργή στην παραγωγή. Τώρα μπορείτε να διαχειρίζεστε τις κρατήσεις του Hotelbeds απευθείας μέσω του BookersDesk.

Δεκέμβριος 17, 2024

Νέο Frontend για το BookersDesk Εκκίνησε στην Παραγωγή

Εκκινήσαμε έναν νέο και βελτιωμένο σχεδιασμό για το BookersDesk, προσφέροντας μια πιο απλή και φιλική προς τον χρήστη διεπαφή για τη διαχείριση ακινήτων.

Δεκέμβριος 2, 2024

Ταμείο για Ακίνητα

Μια νέα λειτουργικότητα για τη διαχείριση των μετρητών είναι τώρα διαθέσιμη, επιτρέποντας μια πιο ασφαλή και ακριβή διαχείριση των χρηματικών συναλλαγών.

Νοέμβριος 15, 2024

Το Hostelworld Στέλνει Κρατήσεις με Στοιχεία Πιστωτικής Κάρτας

Οι κρατήσεις από το Hostelworld περιλαμβάνουν πλέον τα στοιχεία της πιστωτικής κάρτας, διευκολύνοντας τη διαδικασία πληρωμών και τη διαχείριση των κρατήσεων.

Νοέμβριος 6, 2024

Ενσωμάτωση Airbnb API

Έχουμε ενσωματώσει με επιτυχία την Airbnb API, επιτρέποντάς σας να διαχειρίζεστε τις κρατήσεις του Airbnb απευθείας από το BookersDesk.

Οκτώβριος 16, 2024

Προσθήκη Προσφορών για Εμφάνιση Διαφορετικών Τιμών στην Μηχανή Κρατήσεων

Τώρα μπορείτε να προσθέτετε ειδικές προσφορές που εμφανίζουν διαφορετικές τιμές στην μηχανή κρατήσεων, βοηθώντας σας να τρέξετε προωθητικές ενέργειες και να προσελκύσετε περισσότερους επισκέπτες.

Αύγουστος 2, 2024

Σελίδα Κρατήσεων για Ενήλικες & Παιδιά / Προσθήκη Νέας Κράτησης για Συμμετέχοντες

Η σελίδα κρατήσεων επιτρέπει τώρα ευκολότερη επιλογή ενήλικων και παιδικών επισκεπτών, με βελτιώσεις στη λειτουργικότητα της δυνατότητας Προσθήκης Νέας Κράτησης.

Αύγουστος 30, 2024

Πιστοποίηση PCI DSS

Είμαστε στην ευχάριστη θέση να ανακοινώσουμε ότι το BookersDesk είναι τώρα πιστοποιημένο με PCI DSS, εξασφαλίζοντας τα υψηλότερα πρότυπα ασφάλειας για τους χρήστες και τους πελάτες μας.

Ιούλιος 24, 2024

Πρώτα Ακίνητα Προστέθηκαν στο BookersDesk

Έχουμε προσθέσει με επιτυχία τα πρώτα ακίνητα στο BookersDesk, σηματοδοτώντας ένα σημαντικό βήμα στην επέκταση της πλατφόρμας.

Ιούλιος 22, 2024

Προστέθηκαν Νέα Αντικείμενα Upsell

Προστέθηκαν νέα αντικείμενα upsell, επιτρέποντας στα ακίνητα να προσφέρουν περισσότερες επιπλέον επιλογές στους επισκέπτες και να αυξήσουν τα έσοδα.

Ιούλιος 15, 2024

Νέα Αναφορά - Μηνιαία Κατάληψη, Άφιξες και Αναφορά Υποδοχής

Προστέθηκε μια νέα δυνατότητα μηνιαίας αναφοράς για την παρακολούθηση της κατάληψης, των αφίξεων των επισκεπτών και της δραστηριότητας υποδοχής, βοηθώντας σας να έχετε τον έλεγχο των βασικών μετρήσεων.

Ιούλιος 8, 2024

Ενσωμάτωση Hostelworld API

Η Hostelworld API είναι τώρα ενσωματωμένη, επιτρέποντας την απρόσκοπτη διαχείριση των κρατήσεων που γίνονται μέσω του Hostelworld.
